Splunkstr

This popup can help you save Splunk queries to Splunkstr where only one field vary.
Its a simple proces.
1.) Grab the query using the "Grab query button".
2.) Mark your variable using $$$ on either side.
3.) Name your query.
4.) Save using the "Save query button".
Ex. Grab and edit long_splunk_query_containing_my_variable_user_name_rest_of_query.
The query then looks like long_splunk_query_containing_my_variable_$$$user_name$$$_rest_of_query.
Finally press "Save query".
To manage your queries for Splunkstr go to chrome://extensions/ and click on "Options".
